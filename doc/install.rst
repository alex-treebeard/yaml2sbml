Installation
===============================================


To install go to the main folder and do:

```pip install .```


Requirements for users
---------------------------

 - Python 3.6+
 - python-libsbml>=5.18.0
 - PyYAML>=5.1
 - pandas>=1.0.1
 - petab>=0.1.4

Extras (to run examples):
 - amici>=0.11.10
 - numpy>=1.19.4
 - matplotlib>=3.1.0



Requirements for developers
--------------------------------------

Can be found in `requirements.txt`.
